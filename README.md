# Personal Knowledge Vault (Assignment B13-A2)

Welcome to the **Personal Knowledge Vault** project! This is a landing page built to present a solution for structured note-taking and knowledge retention.

## 📖 Description

"Personal Knowledge Vault isn't another note-taking app. It's a thinking environment where every note, link, and summary becomes part of an interconnected web of understanding."

It serves as a private space to store notes, links, and learning summaries, designed to help you think clearly, connect ideas, and turn learning into long-term knowledge. The project highlights that "Notes Don't Fail, Structure Does" and aims to resolve common issues like scattered notes, bookmark overload, isolated ideas, and forgotten knowledge.

## ✨ Features

Based on the project's design and structure, the vault provides the following solutions:

- **Structured Capture:** Capture ideas with structure built-in from day one.
- **Automatic Connections:** Connect related concepts automatically as you write.
- **Reflection Prompts:** Build lasting knowledge through reflection prompts.
- **Semantic Search:** Find anything instantly with semantic search.

## 🛠 Technologies Used

- **HTML5:** Semantic structure and content.
- **CSS3:** Styling, layout (Flexbox/Grid), and typography (Inter font family).

## 🚀 How to Run Locally

To view this project locally, simply follow these steps:

1. Clone the repository or download the source files.
2. Navigate to the project directory.
3. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Safari).
   - Alternatively, you can use a local server like Live Server (VS Code extension) or Python's HTTP server:
     ```bash
     python3 -m http.server
     ```
     Then, open `http://localhost:8000` in your browser.
