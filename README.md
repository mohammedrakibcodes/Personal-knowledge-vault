# Personal Knowledge Vault

A modern and responsive landing page for a personal knowledge management platform.
Built using HTML and CSS with a clean UI focused on structured learning, note organization, and knowledge retention.

## Live Demo

🔗 Live Site: https://mohammedrakibcodes.github.io/B13-A1/

## Features

* Responsive modern landing page
* Clean navigation bar
* Hero section with CTA buttons
* Problem/solution sections
* Feature cards layout
* Second brain knowledge section
* Early access CTA section
* Footer with social links
* Smooth hover effects
* Google Fonts integration
